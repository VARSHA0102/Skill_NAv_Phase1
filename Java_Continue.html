Java Continue Statement

Learn how to use the continue statement in Java with examples and detailed explanations.

What is the Continue Statement?

The continue statement in Java is used to skip the current iteration of a loop and proceed to the next iteration. It is commonly used when you want to bypass specific conditions within a loop without terminating the loop entirely.

Continue Statement in For Loop

The continue statement is often used in for loops to skip certain iterations based on a condition.

Example 1: Using Continue in a For Loop

The following example demonstrates the use of the continue statement in a for loop:

public class ContinueExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            if (i == 5) {
                continue;  // Skip the iteration when i equals 5
            }
            System.out.println(i);
        }
    }
}

Explanation:

In this example:

  • The loop starts from 1 and goes up to 10.
  • When the value of i reaches 5, the continue statement is executed, and the loop skips the rest of the code in that iteration.
  • As a result, the number 5 is not printed, and the loop continues with the next iteration.

Continue Statement in While Loop

The continue statement can also be used in while loops to skip the current iteration and move to the next one.

Explanation:

In this example:

  • The loop starts from 1 and goes up to 10.
  • When the value of i reaches 5, the continue statement is executed, skipping the rest of the code in that iteration.
  • The value 5 is not printed, and the loop continues with the next iteration.

Continue Statement in Nested Loops

The continue statement can also be used in nested loops to skip the current iteration of the inner loop while continuing the outer loop.

Example 3: Continue in Nested Loops

In this example, the continue statement is used within nested loops:

public class NestedLoopContinueExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 3; i++) {
            for (int j = 1; j <= 3; j++) {
                if (i == 2 && j == 2) {
                    continue;  // Skip the iteration when i equals 2 and j equals 2
                }
                System.out.println("i = " + i + ", j = " + j);
            }
        }
    }
}

Explanation:

In this example:

  • The outer loop runs for i from 1 to 3, and the inner loop runs for j from 1 to 3.
  • When i equals 2 and j equals 2, the continue statement is executed, skipping the iteration of the inner loop.
  • The output will skip the case where i = 2 and j = 2.

Conclusion

The continue statement in Java is a useful tool that allows you to skip certain iterations of a loop based on specific conditions. It helps in making your loops more flexible and efficient by enabling you to bypass unnecessary code within loops.

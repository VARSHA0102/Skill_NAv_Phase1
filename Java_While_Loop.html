Java While Loop

Learn how to use the while loop in Java for repeated execution of code blocks.

What is a While Loop?

A while loop in Java is a control flow statement that allows code to be executed repeatedly based on a given Boolean condition. The loop continues to execute as long as the condition remains true.

Syntax of While Loop

while (condition) {
    // code to be executed
}

Example 1: Basic While Loop

Here’s a simple example of a while loop that prints numbers from 1 to 5:

public class WhileLoopExample {
    public static void main(String[] args) {
        int i = 1;
        while (i <= 5) {
            System.out.println(i);
            i++;
        }
    }
}

Explanation:

In the example above:

  • Condition: i <= 5; - The loop will run as long as i is less than or equal to 5.
  • Code Block: System.out.println(i); i++; - Inside the loop, i is printed and then incremented by 1.

Example 2: While Loop with User Input

The following example demonstrates how to use a while loop to keep asking the user for input until they enter a specific value:

import java.util.Scanner;

public class WhileLoopUserInput {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int number = 0;

        while (number != 5) {
            System.out.print("Enter a number (5 to exit): ");
            number = scanner.nextInt();
        }

        System.out.println("You entered 5. Exiting...");
    }
}

Explanation:

In this example:

  • The while loop continues to ask the user for a number until they enter 5.
  • When the user enters 5, the loop exits and a message is displayed.

Example 3: Infinite While Loop

Here’s an example of an infinite while loop:

public class InfiniteWhileLoop {
    public static void main(String[] args) {
        while (true) {
            System.out.println("This loop will run forever...");
        }
    }
}

Explanation:

In this example, the condition is always true (while (true)), so the loop will run indefinitely until the program is terminated manually.

Conclusion

The while loop is a fundamental control structure in Java, allowing you to execute a block of code repeatedly as long as a given condition is true. It's particularly useful when the number of iterations is not known beforehand.

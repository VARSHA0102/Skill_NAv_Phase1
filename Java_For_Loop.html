Java For Loop

Understand how to use the for loop to iterate over a range of values in Java.

What is a For Loop?

A for loop in Java is a control flow statement that allows code to be executed repeatedly based on a given condition. It is particularly useful when the number of iterations is known before entering the loop.

Syntax of For Loop

for (initialization; condition; update) {
    // code to be executed
}

Example 1: Basic For Loop

Hereâ€™s a simple example of a for loop that prints numbers from 1 to 5:

public class ForLoopExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println(i);
        }
    }
}

Explanation:

In the example above:

  • Initialization: int i = 1; - The loop starts with i initialized to 1.
  • Condition: i <= 5; - The loop will run as long as i is less than or equal to 5.
  • Update: i++ - After each iteration, i is incremented by 1.

Example 2: For Loop with Array

The following example demonstrates how to use a for loop to iterate over an array:

public class ForLoopArray {
    public static void main(String[] args) {
        int[] numbers = {10, 20, 30, 40, 50};

        for (int i = 0; i < numbers.length; i++) {
            System.out.println(numbers[i]);
        }
    }
}

Explanation:

In this example, the for loop iterates over an array of integers, printing each element of the array:

  • Initialization: int i = 0; - The loop starts with i initialized to 0, the index of the first element in the array.
  • Condition: i < numbers.length; - The loop runs as long as i is less than the length of the array.
  • Update: i++ - After each iteration, i is incremented by 1.

Example 3: Nested For Loop

The following example shows how to use a nested for loop to print a multiplication table:

public class NestedForLoop {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            for (int j = 1; j <= 5; j++) {
                System.out.print(i * j + "\t");
            }
            System.out.println();
        }
    }
}

Explanation:

In this example:

  • The outer loop iterates over the rows, and the inner loop iterates over the columns.
  • The product of i and j is printed, creating a multiplication table.

Conclusion

The for loop is a versatile and commonly used control structure in Java. It provides a clean and concise way to iterate over a range of values, arrays, and even complex data structures.

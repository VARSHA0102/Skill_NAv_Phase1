Java Keywords

Learn about the reserved keywords in Java and their significance.

What are Java Keywords?

In Java, keywords are reserved words that have a predefined meaning in the Java programming language. These keywords cannot be used as identifiers (names for variables, methods, or classes) because they are integral to the syntax of the language.

List of Java Keywords

Java has 53 reserved keywords. Here is a list of commonly used Java keywords along with their purpose:

  • abstract: Used to declare an abstract class or method.
  • assert: Used for debugging purposes to make an assertion.
  • boolean: Used to declare a variable of type boolean (true or false).
  • break: Exits from a loop or switch statement.
  • byte: Used to declare a variable of type byte (8-bit integer).
  • case: Defines a branch in a switch statement.
  • catch: Used to handle exceptions in a try-catch block.
  • char: Used to declare a variable of type char (single 16-bit Unicode character).
  • class: Used to declare a class.
  • const: Reserved but not used.
  • continue: Skips the current iteration of a loop and proceeds to the next iteration.
  • default: Specifies the default block of code in a switch statement.
  • do: Used to execute a block of code once and then repeat the execution while a condition is true.
  • double: Used to declare a variable of type double (double-precision 64-bit floating point).
  • else: Specifies a block of code to execute if the condition in an if statement is false.
  • enum: Defines a set of named constants.
  • extends: Indicates that a class is inheriting from a superclass.
  • final: Used to declare constants, prevent method overriding, and inheritance.
  • finally: Executes a block of code after a try-catch block, regardless of whether an exception was thrown.
  • float: Used to declare a variable of type float (single-precision 32-bit floating point).
  • for: Used to start a for loop.
  • goto: Reserved but not used.
  • if: Used to start an if statement.
  • implements: Indicates that a class implements an interface.
  • import: Imports other classes or entire packages.
  • instanceof: Tests whether an object is an instance of a specific class or implements an interface.
  • int: Used to declare a variable of type int (32-bit integer).
  • interface: Defines an interface.
  • long: Used to declare a variable of type long (64-bit integer).
  • native: Specifies that a method is implemented in native code using JNI (Java Native Interface).
  • new: Creates new objects.
  • null: Represents a null reference.
  • package: Defines a namespace for organizing classes and interfaces.
  • private: Specifies that a member of a class is accessible only within the class.
  • protected: Specifies that a member of a class is accessible within the same package and subclasses.
  • public: Specifies that a member of a class is accessible from any other class.
  • return: Exits from a method and optionally returns a value.
  • short: Used to declare a variable of type short (16-bit integer).
  • static: Indicates that a member belongs to the class, rather than to instances of the class.
  • strictfp: Used to restrict floating-point calculations to ensure portability.
  • super: Refers to the superclass of the current object.
  • switch: Starts a switch statement to handle multiple possible execution paths.
  • synchronized: Ensures that a method or block of code is accessed by only one thread at a time.
  • this: Refers to the current object.
  • throw: Used to explicitly throw an exception.
  • throws: Indicates the exceptions that a method might throw.
  • transient: Prevents serialization of a member variable.
  • try: Starts a block of code that will be tested for exceptions.
  • void: Indicates that a method does not return a value.
  • volatile: Indicates that a variable may be changed by multiple threads.
  • while: Starts a while loop that executes as long as a condition is true.

Examples

public class KeywordExamples {
    public static void main(String[] args) {
        // Example of using 'class'
        class Example {
            int number;

            // Example of using 'public' and 'void'
            public void displayNumber() {
                System.out.println("Number: " + number);
            }
        }

        // Example of using 'new' and 'instanceof'
        Example example = new Example();
        example.number = 10;

        if (example instanceof Example) {
            example.displayNumber();
        }
    }
}

Conclusion

Java keywords form the basic building blocks of the Java language. They are reserved by the language and cannot be used for any other purpose. Understanding these keywords is essential for writing correct and effective Java programs.

Java Do-While Loop

Understanding the do-while loop in Java with examples and explanations.

What is a Do-While Loop?

A do-while loop is similar to a while loop, but with one key difference: the code block inside the loop is executed at least once before the condition is tested.

Syntax of Do-While Loop

do {
    // code to be executed
} while (condition);

Example 1: Basic Do-While Loop

Here’s a simple example of a do-while loop that prints numbers from 1 to 5:

public class DoWhileLoopExample {
    public static void main(String[] args) {
        int i = 1;
        do {
            System.out.println(i);
            i++;
        } while (i <= 5);
    }
}

Explanation:

In the example above:

  • Code Block: System.out.println(i); i++; - This code is executed first.
  • Condition: i <= 5 - After executing the code block, the condition is checked. If true, the loop runs again.

Example 2: Do-While Loop with User Input

The following example demonstrates a do-while loop that keeps asking the user for input until they enter a specific value:

import java.util.Scanner;

public class DoWhileLoopUserInput {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int number;

        do {
            System.out.print("Enter a number (5 to exit): ");
            number = scanner.nextInt();
        } while (number != 5);

        System.out.println("You entered 5. Exiting...");
    }
}

Explanation:

In this example:

  • The code block inside the do runs first, asking the user for a number.
  • The condition is then checked. If the number is not 5, the loop runs again.
  • When the user enters 5, the loop exits.

Example 3: Infinite Do-While Loop

Here’s an example of an infinite do-while loop:

public class InfiniteDoWhileLoop {
    public static void main(String[] args) {
        do {
            System.out.println("This loop will run forever...");
        } while (true);
    }
}

Explanation:

In this example, the condition is always true (while (true)), so the loop will run indefinitely until the program is manually terminated.

Conclusion

The do-while loop ensures that the code block is executed at least once before the condition is checked. This makes it useful in scenarios where the loop's code must be executed at least once, regardless of the condition.
